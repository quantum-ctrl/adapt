# Contributing to ADAPT

Thank you for your interest in contributing to ADAPT! We welcome contributions from the community to help make this tool better for everyone.

## How to Contribute

1.  **Fork the Repository**: Click the "Fork" button on the top right of the GitHub page to create your own copy of the repository.
2.  **Clone your Fork**:
    ```bash
    git clone https://github.com/quantum-ctrl/adapt.git
    cd ADAPT
    ```
3.  **Create a Branch**: Create a new branch for your feature or bug fix.
    ```bash
    git checkout -b feature/my-new-feature
    ```
4.  **Make Changes**: Implement your changes and ensure they work as expected.
5.  **Commit Changes**:
    ```bash
    git commit -m "Add a descriptive message about your changes"
    ```
6.  **Push to your Fork**:
    ```bash
    git push origin feature/my-new-feature
    ```
7.  **Submit a Pull Request**: Go to the original repository on GitHub and open a Pull Request from your branch.

## Reporting Issues

If you find a bug or have a suggestion for a new feature, please open an issue in the standard GitHub Issue Tracker.

Please include:
*   A clear description of the issue or feature request.
*   Steps to reproduce the issue (if applicable).
*   Your operating system and Python version.

## Coding Standards

*   Please follow PEP 8 style guidelines for Python code.
*   Keep code clean and well-documented.
